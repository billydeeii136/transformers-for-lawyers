# Harvey Devtool Integrations
This directory contains integration templates for:
- Warp (`warp/`)
- VS Code + GitHub Copilot (`vscode/`)
- Codex (`codex/`)
- ChatGPT Responses API MCP usage (`chatgpt/`)
- Xcode (`xcode/`)

Use `.harvey/installers/devtools/install_devtool_integrations.sh` to package/apply these templates.
